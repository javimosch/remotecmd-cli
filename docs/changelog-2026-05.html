v1.0.0 May 2026

Initial Release

remotecmd-cli is born — remote command execution via WebSocket relay.

🚀

Run commands on remote machines

Execute shell commands on any connected machine and get structured JSON output back — stdout, stderr, exit code, and duration. No SSH, no VPN, no open ports required.

🔀

WebSocket relay hub

A lightweight always-on relay routes commands between clients and target daemons. Deploy it on any VPS — all traffic flows through it, so neither side needs inbound firewall rules.

🔑

Token-based auth

Each target machine gets a unique secret token. Clients must present the correct token to execute commands, so only authorised callers can reach your machines.

⌨️

Convenience aliases

Install rc, rcx, rcl, and rcs aliases for faster day-to-day usage. rcx myserver 'uptime' is all you need.
